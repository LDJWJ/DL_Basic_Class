- 의사결정트리 및 앙상블 기법 

(1) 선형회귀모델
  - 단항 선형회귀
  - 다항 선형회귀 
  
(2) 선형회귀모델 일반화 방법 - w의 계수를 제한
  - 릿지(Ridge) - L2 규제 - w값을 0에 가깝게 만든다. 변수는 남겨둔다. 
  - 랏소(Lasso) - L1 규제 - w값을 0에 가깝게 만든다. 변수를 없앤다.
  
(3) 선형모델중에 이진분류
  - 로지스틱 회귀(Logistic)
  - 서포트 벡터 머신(SVM)
  
(4) 멀티클래스 분류에 로지스틱 회귀를 사용한다.
  - 일대다을 이용할 경우에 클래스가 5개이면 이진분류 모델을 몇개 만든다. 5개의 이진분류 모델을 생성.
  - 일대일
  
  -> 그중에 예측 점수가 좋은 모델을 사용한다.
  -> knn, decision tree도 이용 가능.
  
(5) 의사결정트리는 단점 
  -> 과대적합이 일어난다.
  
(6) 과대적합이 일반화하는 방법을 하나 배웠는데...
